# Interview Preparation

## Recommended Study Pattern

For each day, follow this rhythm:

1. 30 minutes — refresh the theory
2. 45 minutes — go deeper into internals and edge cases
3. 45 minutes — answer 20–30 interview questions aloud
4. 30 minutes — solve one coding or design problem

## Day 01 — JVM & Memory

**Week 1 — Java Mastery**

**Theory**

- JVM architecture
- Stack vs heap
- Metaspace
- Stack frames
- Class loading
- Java Memory Model

**Practice**

- 25 interview questions
- Review memory-related bugs and GC behavior

**Theory**

## Final Advice

Since you already have strong practical experience, prioritize:

- internals over basics
- clear explanations over memorization
- real-world examples over textbook answers
- structured practice over passive reading
